# Xtract

Xtract is an IntelliJ IDEA plugin that recommends **Extract Method**
refactorings using Artificial Intelligence and Explainable AI.

The plugin follows the **W3B recommendation criteria**, providing developers
with recommendations that explain:

- **Which** code fragment should be extracted;
- **Where** the new method should be placed;
- **Why** the refactoring is recommended;
- **Benefits** expected after the refactoring.

---

## Features

- IntelliJ IDEA plugin
- Automatic detection of Extract Method opportunities
- Code fragment ranking using CodeBERT
- Prediction using Random Forest
- Explainability using SHAP, LIME, and ANCHOR
- Natural language recommendations generated with GPT
